# DateWidget

A macOS desktop widget built with SwiftUI. Shows the current date, a multi-timezone clock, and a monthly calendar — all wrapped in the macOS 26 Liquid Glass UI.

## Features

- Current date and large day number
- Clock row: local time + New York time
- Monthly calendar grid with today highlighted
- Liquid Glass design (requires macOS 26 Tahoe)

## Requirements

- macOS 26 Tahoe or later
- Xcode 26 or later

## Running

Open `DateWidget.xcodeproj` in Xcode and hit **Run**.
